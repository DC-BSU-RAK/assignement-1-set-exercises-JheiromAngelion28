HTML & CSS Quick Reference Guide

By Jheirom M. Pablo

HTML Tags

<h1> - <h6>: Headings

Defines HTML headings, with <h1> being the largest and <h6> the smallest.

<h1>This is a heading</h1>

<p>: Paragraph

This defines a paragraph of text.

<p>This is a paragraph.</p>

<a>: Anchor (Link)

Creates a hyperlink to another webpage.

<a href="https://www.google.com">Visit Example</a>

<img>: Image

Embeds an image in the webpage. The "Alt" may be needed if the picture fails to load or corrupted.

<img src="image.jpg" alt="Description of image">

<ul>, <ol>, <li>: Lists

Defines unordered and ordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<table>: Table

Defines a table structure.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

CSS Properties

color

Defines the text color.

p {
    color: blue;
}

background-color

Sets the background color of an element.

body {
    background-color: lightgray;
}

font-size

Sets the size of the text.

h1 {
    font-size: 24px;
}

margin

Defines space around an element.

div {
    margin: 20px;
}

padding

Defines space inside an element, between its content and border.

button {
    padding: 10px 20px;
}

border

Defines the border around an element.

p {
    border: 1px solid black;
}

display

Specifies the display behavior of an element.

span {
    display: block;
}

text-align

Defines the horizontal alignment of text.

h2 {
    text-align: center;
}
